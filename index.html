Getting started

Everything you need to run your first search.

What is an API Key?

A free Google credential that lets Jota's Brain query public YouTube data on your behalf. Without it, no search can run. Everything stays in your browser.

How to get one

1. Go to console.cloud.google.com
2. Create a new project
3. Navigate to APIs & Services → Library
4. Enable the YouTube Data API v3
5. Go to Credentials → Create Credentials → API key
6. Paste it into the API Keys panel.

Quota & limits

The free tier gives 10,000 units/day per key, resetting at midnight Pacific. A 30-channel search uses roughly 1,200 units. Deep search uses 3–6× more — use it when regular search misses results with tight filters.

Multiple keys

Add each key from a separate Google Cloud project. When one runs out, the next is used automatically.
